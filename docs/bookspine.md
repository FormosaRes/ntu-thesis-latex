# 📏 書背（側邊）怎麼做<br>How to make the book spine

書背是論文裝訂後印在**書側**的那條字。規範 §2 有規定內容，
但沒人教你**寬度怎麼抓**、**檔案怎麼做** —— 這頁把兩件事都講清楚。

## 📐 寬度怎麼抓（最多人卡住的地方）

**書背寬度 = 論文裝訂後的厚度。** 這個數字**沒有標準值**，取決於你的頁數、
紙張磅數、單／雙面印刷、平裝或精裝。

### 估算方式

內頁用規範指定的 **A4 80 磅模造紙**，厚度約 **0.1 mm／張**：

```
書背寬度 ≈ 張數 × 0.1 mm + 封面封底厚度
```

| 情況 | 張數 | 估計厚度 |
|---|---|---|
| 100 頁雙面印刷 | 50 張 | 約 5 mm |
| 150 頁雙面印刷 | 75 張 | 約 8 mm |
| 200 頁雙面印刷 | 100 張 | 約 10 mm |
| 100 頁單面印刷 | 100 張 | 約 10 mm |

再加平裝封面（200 磅銅西卡）約 **0.5–1 mm**；精裝本更厚，通常 **由裝訂廠量**。

### ✅ 最可靠的做法

**先把內文印出來、拿去裝訂廠，請他們量厚度**，再依實際數字做書背。
很多印刷廠本來就提供這個服務，也常會直接幫你排書背 ——
**印之前先問一句，可以省下重做一次的錢。**

> 💡 題目太長怎麼辦？本範本的 LaTeX 版會**自動縮小字級**以塞進書背高度，
> 不會裁切。Word 版則需自己調整字級。

## 🛠️ 三種做法

### 做法 1：Word（不用 LaTeX）

📥 **下載 → [`ntu-bookspine.docx`](../word/ntu-bookspine.docx)**

1. 用 Word 打開，把 `【】` 裡的字換成自己的
2. 依實際厚度調整表格**欄寬**（範本預設 3 cm 只是示意）
3. **印刷前刪除灰色框線**（那只是標示書背範圍）

適合：只需要書背、不想碰 LaTeX 的人。

### 做法 2：LaTeX（本 repo 範本）

書背是獨立檔案，可單獨編譯：

```bash
xelatex bookspine.tex        # 產出 bookspine.pdf
```

論文資訊會自動從 `main.tex` 的 `\ntusetup{...}` 帶入，不必重打。
要調整書背高度，改 `\spineheight`（預設 25 cm）：

```latex
\setlength{\spineheight}{25cm}
```

在 Overleaf 上編譯書背：`Menu ▸ Settings ▸ Main document ▸ bookspine.tex`
（編完記得改回 `main.tex`）→ 詳見 [overleaf.md](overleaf.md)

### 做法 3：交給裝訂廠

把論文題目、姓名、系所、年月寫在紙上交給廠商，多數印刷廠會幫你排。
**精裝本（碩士紅底燙金／博士黑底燙金）本來就得由廠商燙金**，
書背通常一併處理。

## ⚠️ 送印前檢查

```
□ 書背寬度是量過的，不是猜的
□ 六項內容齊全（校名／系所／學位／中文題目／姓名／年月）
□ 只放中文題目，沒有放英文題目
□ 示意框線已刪除
□ 直式方向正確（由上而下閱讀）
□ 精裝本：碩士紅底燙金、博士黑底燙金已跟廠商確認
```
