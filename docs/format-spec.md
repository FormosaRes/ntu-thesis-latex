書背（側邊）要放：**校名、系所、學位、論文中文題目、撰者姓名、提出年月**。

## 📑 論文次序

必須照這個順序裝訂：

```
封面（含側邊）
書名頁（內容同封面）
論文口試委員會審定書
序言或謝辭          ← 自行決定是否撰寫
中文摘要及關鍵詞 5–7 個
英文摘要及關鍵詞 5–7 個
目次
圖次
表次
論文正文
參考文獻
附錄
封底
```

## 🖨️ 版面與字體（§10）

| 項目 | 規定 |
|---|---|
| 邊界（內文） | 上 3 cm、下 2 cm、**左右各 3 cm** |
| 中文字體 | **12 號楷書**（細明體及標楷體為主）|
| 英文字體 | **12 號 Times New Roman** |
| 中文行距 | **1.5 倍** |
| 英文行距 | **雙行距** |
| 頁碼 | 各頁**正下方置中** |
| 字體顏色 | 黑色；全文不得塗汙刪節、不得使用複寫紙 |

> 封面的上留白是 **4 cm**，與內文的 3 cm 不同。

## 📝 各部分要求

| 部分 | 要求 |
|---|---|
| 謝辭 | 須另頁書寫；內容簡要，**以不超過一頁為原則** |
| 中文摘要 | **不超過三頁**；應含論述重點、方法或程序、結果與討論及結論 |
| 英文摘要 | **不超過三頁** |
| 關鍵詞 | 中英文各 **5–7 個** |
| 目次 | 各章節標題、附錄及所在頁數 |
| 圖次／表次 | 各章節圖表及頁數；**擷取自參考文獻者須標註來源** |
| 參考文獻 | 須含作者姓氏、出版年次、技術資料或期刊名稱、頁碼 |

> 經系所同意以英文撰寫論文者，**仍需附中文摘要**。

## 📚 印製與裝訂（§11–12）

| 項目 | 規定 |
|---|---|
| 印刷面 | **雙面印刷為原則**；80 頁以下得單面（彩色圖片亦可單面）|
| 內頁紙張 | 白色 A4 **80 磅**模造紙 |
| 平裝本封面 | 淺色 **200 磅**銅西卡紙或雲彩紙（上亮 P），A4 |
| 精裝本封面 | **碩士班紅底燙金字；博士班黑底燙金字**，A4 |

平裝本或精裝本**不拘**。

## 📦 要繳幾冊

| 系所 | 冊數 | 繳交地點 |
|---|---|---|
| 一般系所 | **2 冊** | 總圖書館 |
| 數學、物理、化學、海洋、社會科學院、法律學院各系所 | **3 冊** | 總圖書館 |
| 醫學院、公衛學院 | **2 冊** | 醫學圖書館 |

> ⚠️ 這張表是**混合制**：前四項是**逐一點名的系所**，後兩項才是**整個學院**。
> 不能因為數學／物理／化學／海洋屬理學院，就推論「理學院都要 3 冊」——
> 理學院其他系所（地質科學系、心理系、大氣科學系等）**未被點名，適用 2 冊**。
> 冊數**含代繳國家圖書館 1 冊**。

## ⚖️ 系所可以另訂（§13）

> 各系（所、學位學程）得依其學術領域之特殊性另訂統一格式，
> **惟主體架構仍請依本規範訂定。**

所以：**版面主體照校規，細節問系所助教。**
